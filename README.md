# Traps

Traps is a **2D mobile platformer** developed in **Godot** as a **team project** for a mobile application development course.  
The game focuses on **timing, precision, and trap-based challenges**, where players must navigate hazardous levels to reach the goal.

## Features

- Smooth 2D platformer movement and jumping  
- Trap-based hazards and timing challenges  
- Multiple levels with increasing difficulty  
- Simple and clear user interface  
- Designed for mobile gameplay  

---

## Tech Stack

- **Engine:** Godot  
- **Language:** GDScript  
- **Platform:** Mobile (Android / iOS)  

---

## How to Run

1. Install **Godot** (version X.X or later)  
2. Clone or download this repository  
3. Open the project in Godot  
4. Press **Play** to run the game
